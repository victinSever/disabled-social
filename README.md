# 残疾人婚恋APP

### 项目简介
残疾人婚恋交友平台，提供残疾人用户一个集交友，分享，聚会，咨询，就业，婚恋等功能的一个社交平台。

#### 前端技术
- uniApp框架
- axios请求交互
